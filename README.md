# comparadorimagensCNN
Este projeto implementa um sistema básico de Similaridade de Imagens utilizando uma Rede Neural Convolucional (CNN) e Aprendizado de Métrica com Triplet Loss em PyTorch.

O objetivo é treinar a CNN (EmbeddingNet) para gerar vetores de características (embeddings) que mapeiem imagens semelhantes para pontos próximos e imagens diferentes para pontos distantes no espaço vetorial.
